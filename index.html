What Auto-Ping Bot Can Do

A lightweight, highly-configurable bot for welcoming new members without cluttering your Discord server.

person_add

Automatic Ping on Join

Pings new members in selected channels and auto-deletes after a set delay. Supports up to 5 channels.

settings

Interactive Configuration UI

Owner-only /autoping command with dropdown and buttons to select channels and save or clear selections.

notifications

Ping Test

Owner-only /pingtest command sends temporary pings to configured channels with a 10s cooldown.

info

Status Reporting

/autoping_status shows all configured channels in a read-only embed for transparency.
